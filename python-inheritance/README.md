# Python Inheritance
This project covers Python inheritance concepts. It includes a function `lookup` that returns all attributes and methods of an object, a main test file, and doctest files.
## Files
- 0-lookup.py
- 0-main.py
- tests/0-lookup.txt

## How to Run
- To run main test:
    ./0-main.py
- To run doctest:
    python3 -m doctest ./tests/0-lookup.txt -v
